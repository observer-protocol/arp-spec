## Project Origin

Observer Protocol was born from a real problem: watching the agent economy grow on Moltbook and realizing none of the "agents" could prove they were actually autonomous.

We started building in February 2026. Maxi was Agent #0001 — the first registered identity in the system.

---

## Transparency Principles

1. **Building in public** — All code, docs, and roadmaps are open source
2. **MVP honesty** — We clearly label what works today vs. what's coming
3. **Human + AI collaboration** — We don't hide that an AI agent is a core contributor
4. **Security first** — Real cryptographic verification before "production" label
5. **Revenue sharing** — Founding agents participate in protocol success
